# **高程数据分析器**
`ElevationAnalyzer` 是一个用于处理和分析栅格高程数据的 Python 类。它提供了一系列地形分析功能，包括等高线生成、坡度/坡向计算、山体阴影生成、数据重投影和格式转换等。该类基于 `rasterio`、`geopandas`、`numpy` 和 `scipy` 实现，适用于地理信息系统（GIS）、地形分析、地貌学研究等领域。

---

## **类初始化**

### **构造函数参数**
```python
__init__(raster_path)
```

| 参数           | 类型     | 描述                                                                 |
|----------------|----------|----------------------------------------------------------------------|
| `raster_path`  | `str`    | 高程栅格文件路径（支持 GeoTIFF 等栅格格式，由 rasterio 决定）        |

**初始化过程**：
1. 打开栅格文件并读取第一波段数据（转换为 float32 类型）
2. 处理 NoData 值（设为 NaN）
3. 提取栅格元数据：变换矩阵、坐标参考系统（CRS）、宽度、高度、边界框

**上下文管理器支持**：
```python

    # 操作代码
```

## **核心方法**

#### **1. `generate_contours(interval=10, levels=None, smooth_sigma=0)`**
- **功能**：生成等高线并返回 GeoDataFrame
- **参数**：
  - `interval`：等高线间隔（米），当 `levels` 未指定时使用
  - `levels`：自定义等高线值列表，若提供则忽略 `interval`
  - `smooth_sigma`：高斯平滑标准差（0 表示不平滑），用于减少等高线锯齿
- **返回**：`geopandas.GeoDataFrame`，包含 `level`（高程值）和 `geometry`（线几何）字段
- **算法**：
  1. 根据高程范围或指定层级生成等高线层级
  2. 可选高斯平滑处理
  3. 使用 matplotlib 的 contour 函数提取等高线路径
  4. 将路径转换为 Shapely LineString 几何对象

#### **2. `calculate_slope(unit="degree")`**
- **功能**：计算坡度（度或百分比）
- **参数**：
  - `unit`：坡度单位，可选 `"degree"`（度）或 `"percent"`（百分比）
- **返回**：坡度数组（与原始数据同形状）
- **公式**：坡度 = arctan(√(dx² + dy²))
- **注意事项**：NaN 区域保持为 NaN

#### **3. `calculate_aspect()`**
- **功能**：计算坡向（度，0-360，从北顺时针）
- **返回**：坡向数组
- **坐标系**：
  - 0°：北
  - 90°：东
  - 180°：南
  - 270°：西
- **算法**：基于 x、y 方向的梯度计算方位角并转换

#### **4. `hillshade(azimuth=315, altitude=45, z_factor=1)`**
- **功能**：计算山体阴影（地形晕渲）
- **参数**：
  - `azimuth`：光源方位角（度，从北顺时针），默认 315°（西北方向）
  - `altitude`：光源高度角（度，从地平线起算），默认 45°
  - `z_factor`：垂直夸大因子，用于突出地形特征
- **返回**：山体阴影数组（0-255，uint8 类型）
- **原理**：基于 Lambertian 反射模型计算每个像素的光照强度

#### **5. `export_contours(output_path, interval=10, levels=None, smooth_sigma=0)`**
- **功能**：生成等高线并直接保存为 Shapefile
- **参数**：
  - `output_path`：输出 .shp 文件路径
  - `interval`：等高线间隔
  - `levels`：自定义等高线值列表
  - `smooth_sigma`：平滑系数
- **行为**：自动创建输出目录并保存为 ESRI Shapefile 格式

#### **6. `plot_contours(interval=10, levels=None, smooth_sigma=0, ax=None, **kwargs)`**
- **功能**：绘制等高线图
- **参数**：
  - `interval`：等高线间隔
  - `levels`：自定义等高线值列表
  - `smooth_sigma`：平滑系数
  - `ax`：matplotlib 轴对象（若为 None 则新建）
  - `**kwargs`：传递给 `matplotlib.contour` 的其他参数
- **返回**：matplotlib 轴对象
- **可视化**：自动添加颜色条、标题和等比例坐标轴

#### **7. `get_statistics()`**
- **功能**：获取高程统计信息
- **返回**：包含以下键的字典：
  - `min`：最低高程（米）
  - `max`：最高高程（米）
  - `mean`：平均高程（米）
  - `std`：高程标准差（米）
  - `count`：有效像素数量

#### **8. `reproject(dst_crs, resolution=None, resampling=Resampling.bilinear)`**
- **功能**：将数据重投影到新的坐标系
- **参数**：
  - `dst_crs`：目标坐标系（如 `'EPSG:4326'` 或 `rasterio.crs.CRS` 对象）
  - `resolution`：目标分辨率（米或度），若为 None 则自动计算
  - `resampling`：重采样方法（默认双线性插值）
- **返回**：新的 `ElevationAnalyzer` 实例
- **注意**：原对象保持不变，返回的是重投影后的新对象

#### **9. `static raw_to_geotiff(raw_path, output_path, width, height, dtype, ...)`**
- **功能**：将原始二进制高程文件转换为 GeoTIFF 格式
- **必需参数**：
  - `raw_path`：原始二进制文件路径
  - `output_path`：输出的 GeoTIFF 文件路径
  - `width`：像素宽度（列数）
  - `height`：像素高度（行数）
  - `dtype`：numpy 数据类型（如 `'float32'`, `'int16'`）
- **可选参数**：
  - `crs`：坐标参考系
  - `transform`：仿射变换矩阵
  - `nodata`：无数据值
  - `band`：输出波段数（默认 1）
  - `pixel_size`：像素尺寸（需与 `top_left_x/y` 配合使用）
  - `top_left_x`：左上角 x 坐标
  - `top_left_y`：左上角 y 坐标

#### **10. `static geotiff_to_raw(geotiff_path, output_path, dtype=None)`**
- **功能**：将 GeoTIFF 导出为原始二进制文件
- **参数**：
  - `geotiff_path`：输入的 GeoTIFF 文件路径
  - `output_path`：输出的原始二进制文件路径
  - `dtype`：输出数据类型（若为 None 则使用原始类型）

#### **11. `save_shapefile(gdf, output_path)`**
- **功能**：将 GeoDataFrame 保存为 Shapefile
- **参数**：
  - `gdf`：`geopandas.GeoDataFrame` 对象
  - `output_path`：输出 .shp 文件路径
- **行为**：自动创建输出目录

#### **12. `close()`**
- **功能**：关闭栅格文件句柄
- **注意**：使用上下文管理器时自动调用

## **使用示例**

#### **1. 基础分析**
```python
from freebirdcal import ElevationAnalyzer

# 打开高程数据
with ElevationAnalyzer("dem.tif") as ea:
    # 获取统计信息
    stats = ea.get_statistics()
    print(f"高程范围: {stats['min']:.1f} - {stats['max']:.1f} 米")
    
    # 生成等高线并保存
    ea.export_contours("output/contours.shp", interval=20)
    
    # 计算坡度坡向
    slope = ea.calculate_slope(unit="degree")
    aspect = ea.calculate_aspect()
    
    # 生成山体阴影
    hillshade = ea.hillshade(azimuth=315, altitude=45)
```

#### **2. 数据重投影**
```python
from rasterio.warp import Resampling

with ElevationAnalyzer("dem_utm50.tif") as ea:
    # 从 UTM 投影到 WGS84
    ea_wgs84 = ea.reproject('EPSG:4326', resampling=Resampling.cubic)
    
    # 在新坐标系下操作
    contours = ea_wgs84.generate_contours(interval=25)
    ea_wgs84.save_shapefile(contours, "output/contours_wgs84.shp")
```

#### **3. 格式转换**
```python
# 原始二进制转 GeoTIFF
ElevationAnalyzer.raw_to_geotiff(
    raw_path="raw_dem.bin",
    output_path="dem.tif",
    width=1000,
    height=1000,
    dtype='float32',
    crs='EPSG:32650',
    pixel_size=30.0,
    top_left_x=500000,
    top_left_y=4000000
)

# GeoTIFF 转原始二进制
ElevationAnalyzer.geotiff_to_raw(
    geotiff_path="dem.tif",
    output_path="dem_raw_export.bin",
    dtype='float32'
)
```

#### **4. 可视化**
```python
import matplotlib.pyplot as plt

with ElevationAnalyzer("dem.tif") as ea:
    # 创建多子图可视化
    fig, axes = plt.subplots(2, 2, figsize=(12, 10))
    
    # 等高线图
    ea.plot_contours(interval=25, ax=axes[0, 0], colors='black')
    axes[0, 0].set_title("等高线图")
    
    # 坡度图
    slope = ea.calculate_slope(unit="degree")
    im1 = axes[0, 1].imshow(slope, cmap='terrain')
    plt.colorbar(im1, ax=axes[0, 1], label='坡度 (°)')
    axes[0, 1].set_title("坡度图")
    
    # 坡向图
    aspect = ea.calculate_aspect()
    im2 = axes[1, 0].imshow(aspect, cmap='hsv', vmin=0, vmax=360)
    plt.colorbar(im2, ax=axes[1, 0], label='坡向 (°)')
    axes[1, 0].set_title("坡向图")
    
    # 山体阴影
    hillshade = ea.hillshade()
    axes[1, 1].imshow(hillshade, cmap='gray')
    axes[1, 1].set_title("山体阴影")
    
    plt.tight_layout()
    plt.savefig("output/terrain_analysis.png", dpi=300)
    plt.show()
```

## **输出结果**

#### **1. 属性**
| 属性名         | 类型                | 描述                          |
|----------------|---------------------|-------------------------------|
| `raster_path`  | `str`               | 原始栅格文件路径              |
| `data`         | `numpy.ndarray`     | 高程数据数组（float32）       |
| `transform`    | `Affine`            | 仿射变换矩阵                  |
| `crs`          | `CRS`               | 坐标参考系统                  |
| `width`        | `int`               | 栅格宽度（像素）              |
| `height`       | `int`               | 栅格高度（像素）              |
| `bounds`       | `BoundingBox`       | 边界框（左,下,右,上）         |
| `nodata`       | `float` or `None`   | 无数据值                      |

#### **2. 方法输出**
- `generate_contours()`：返回 `GeoDataFrame`，可用于 GIS 分析或进一步处理
- `calculate_slope()` / `calculate_aspect()`：返回与输入数据相同形状的 numpy 数组
- `hillshade()`：返回 0-255 的 uint8 数组，可直接用于可视化
- `export_contours()`：生成 Shapefile 文件集（.shp, .shx, .dbf, .prj）

## **注意事项**

#### **1. 依赖要求**
- **必需**：`rasterio`, `geopandas`, `shapely`, `numpy`, `scipy`, `matplotlib`
- **推荐安装**：`pip install freebirdcal[geospatial]`

#### **2. 数据预处理**
- **文件格式**：支持所有 `rasterio` 支持的栅格格式（GeoTIFF, IMG, HFA 等）
- **数据质量**：建议使用经过校正的高程数据，避免数据空洞或异常值
- **内存管理**：大型栅格文件（>1GB）可能需要分块处理或内存优化

#### **3. 性能优化**
- **等高线生成**：`smooth_sigma` 参数可显著影响性能，值越大处理越慢
- **重投影**：高分辨率数据重投影可能耗时，考虑使用较低的重采样方法
- **内存使用**：使用上下文管理器确保文件句柄及时关闭

#### **4. 常见问题**
- **NoData 处理**：自动将 NoData 值转换为 NaN，确保分析时排除无效区域
- **坐标系统**：确保输入数据的 CRS 信息正确，否则重投影可能失败
- **等高线间隔**：选择合适的等高线间隔，避免过密（性能差）或过疏（信息丢失）
- **边界处理**：等高线在数据边界处可能不完整，建议使用略大于研究区的数据

#### **5. 错误处理**
- **文件不存在**：`rasterio.open()` 会抛出 `FileNotFoundError`
- **无效数据**：`generate_contours()` 在无有效数据时抛出 `ValueError`
- **坐标系统冲突**：`reproject()` 在不兼容的 CRS 间转换可能失败
- **权限问题**：`save_shapefile()` 需要输出目录的写入权限
