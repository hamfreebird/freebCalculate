## **快速入门**

### 基本用法
```python
from freebirdcal.astro_simulator import AstronomicalSimulator

# 初始化模拟器
sim = AstronomicalSimulator(
    image_size=2048,        # 图像尺寸（像素）
    pixel_scale=0.2,        # 0.2角秒/像素
    zeropoint=25.0,         # AB星等系统零点
    gain=2.0,               # 相机增益（e-/ADU）
    exposure_time=30.0,     # 曝光时间（秒）
    ra_center=180.0,        # 中心赤经（度）
    dec_center=0.0,         # 中心赤纬（度）
    wcs_projection='TAN'    # WCS投影类型
)

# 生成恒星参数（使用幂律分布）
stars = sim.generate_stars(
    num_stars=500,
    min_mag=18,
    max_mag=24,
    distribution='clustered',     # 空间分布：'uniform'或'clustered'
    magnitude_law='powerlaw',     # 星等分布：'uniform'或'powerlaw'
    magnitude_slope=0.33          # 幂律分布斜率
)

# 生成PSF核（Moffat分布，带椭圆率）
psf = sim.generate_psf(
    fwhm=3.0,
    profile='moffat',
    beta=3.5,              # Moffat分布的beta参数
    ellipticity=0.2,       # 椭圆率（0为圆形）
    position_angle=45.0    # 椭圆方向角（度）
)

# 生成图像（包含完整噪声模型）
image = sim.generate_image(
    stars=stars,
    psf_kernel=psf,
    sky_brightness=21.0,
    read_noise=3.0,        # 读出噪声（e- RMS）
    dark_current=0.05,     # 暗电流（e-/像素/秒）
    include_cosmic_rays=True,
    cosmic_ray_rate=0.0005
)

# 生成星表（包含测量误差）
catalog = sim.generate_catalog(stars, include_errors=True)

# 验证模拟结果
validation = sim.validate_simulation(stars, image)

# 保存为FITS文件（支持压缩）
sim.save_to_fits(
    image=image,
    catalog=catalog,
    filename='observation.fits',
    overwrite=True,
    compression=True
)

print(f"模拟完成！恒星数量: {len(stars['x'])}")
```

## **主要参数说明**

### AstronomicalSimulator 初始化参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `image_size` | int | 1024 | 图像的边长（像素），必须为正整数 |
| `pixel_scale` | float | 0.2 | 像素比例（角秒/像素），必须为正数 |
| `zeropoint` | float | 25.0 | AB星等系统零点（m = -2.5*log10(flux) + zeropoint） |
| `gain` | float | 2.0 | 相机增益（e-/ADU），必须为正数 |
| `exposure_time` | float | 1.0 | 曝光时间（秒），必须为正数 |
| `ra_center` | float | 180.0 | 图像中心赤经（度） |
| `dec_center` | float | 0.0 | 图像中心赤纬（度） |
| `wcs_projection` | str | 'TAN' | WCS投影类型，支持：'TAN'（正切）、'SIN'（正弦）、'CAR'（笛卡尔） |

### generate_stars() 参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `num_stars` | int | 100 | 恒星数量，必须为正整数 |
| `min_mag` | float | 18 | 最小星等 |
| `max_mag` | float | 24 | 最大星等，必须≥min_mag |
| `distribution` | str | 'uniform' | 空间分布类型：'uniform'（均匀）、'clustered'（聚类） |
| `magnitude_law` | str | 'uniform' | 星等分布类型：'uniform'（均匀）、'powerlaw'（幂律） |
| `magnitude_slope` | float | 0.33 | 幂律分布斜率（dN/dm ∝ 10^(slope*m)） |

### generate_psf() 参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `fwhm` | float | 2.5 | 全宽半最大值（像素），必须为正数 |
| `profile` | str | 'gaussian' | PSF类型：'gaussian'（高斯）、'moffat'（Moffat） |
| `beta` | float | 4.0 | Moffat分布的beta参数（仅当profile='moffat'时使用），必须>1 |
| `ellipticity` | float | 0.0 | 椭圆率，0为圆形，<1为椭圆 |
| `position_angle` | float | 0.0 | 椭圆方向角（度） |

### generate_image() 参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `stars` | dict | - | 恒星参数字典，来自generate_stars() |
| `psf_kernel` | ndarray | - | PSF卷积核，来自generate_psf() |
| `sky_brightness` | float | 21.0 | 天光背景星等（mag/arcsec²） |
| `read_noise` | float | 5.0 | 读出噪声（e- RMS） |
| `dark_current` | float | 0.1 | 暗电流（e-/像素/秒） |
| `include_cosmic_rays` | bool | False | 是否包含宇宙射线 |
| `cosmic_ray_rate` | float | 0.001 | 宇宙射线率（事件/像素） |

## **新特性与优化**

### 1. 增强的真实性
- **恒星分布**：支持均匀分布和聚类分布
- **星等分布**：支持均匀分布和幂律分布（模拟真实星等分布）
- **PSF模型**：支持椭圆PSF，模拟大气湍流和光学像散
- **噪声模型**：完整的噪声模型包括：
  - 光子噪声（Poisson）
  - 读出噪声（高斯）
  - 暗电流
  - 宇宙射线（可选）
- **WCS坐标系**：完整的WCS支持，包括ICRS参考系和J2000.0历元

### 2. 科学准确性
- **通量计算**：正确考虑曝光时间和增益
- **Moffat PSF**：根据FWHM正确计算alpha参数
- **误差估计**：星表包含测量误差估计
- **通量守恒**：验证PSF卷积后的通量守恒

### 3. 性能优化
- **向量化操作**：使用bincount和向量化方法提高恒星添加效率
- **智能卷积**：根据PSF核大小自动选择直接卷积或FFT卷积
- **边界处理**：完善的边界处理，避免数组越界错误

### 4. 错误处理与验证
- **输入验证**：所有参数都有范围检查和类型验证
- **日志记录**：详细的日志记录，便于调试和监控
- **模拟验证**：`validate_simulation()`方法验证模拟结果的一致性
- **文件操作**：安全的文件保存，检查目录权限和文件覆盖

## **输出文件格式**

### FITS文件结构
```
Primary HDU (图像数据)
├── 图像数据 (ADU单位)
├── WCS头部信息 (完整的天文坐标)
├── 仪器参数 (增益、零点、曝光时间等)
└── 参考系信息 (ICRS, J2000.0)

Bintable HDU (星表)
├── ID: 恒星编号
├── RA, DEC: 赤经赤纬 (度)
├── MAG: 星等
├── FLUX: 通量 (e-/s)
├── FLUX_TOTAL: 总电子数
├── X_PIXEL, Y_PIXEL: 像素坐标
├── MAG_ERR, FLUX_ERR: 测量误差 (可选)
└── SNR: 信噪比 (可选)
```

### 验证输出
`validate_simulation()` 返回的字典包含：
- `num_stars_input`: 输入恒星数量
- `total_flux_input`: 输入总通量 (e-)
- `image_mean`, `image_std`: 图像统计量
- `background_median`, `background_mad`: 背景估计
- `snr_approx`: 近似信噪比

## **高级用法示例**

### 多波段模拟
```python
import numpy as np
from freebirdcal.astro_simulator import AstronomicalSimulator

# 初始化模拟器
sim = AstronomicalSimulator(image_size=1024, pixel_scale=0.4)

# 生成恒星（相同的空间分布）
stars = sim.generate_stars(num_stars=1000)

# 不同波段的PSF和背景
bands = {
    'B': {'psf_fwhm': 2.5, 'sky_brightness': 22.5, 'zeropoint': 25.0},
    'V': {'psf_fwhm': 2.0, 'sky_brightness': 21.5, 'zeropoint': 25.0},
    'R': {'psf_fwhm': 1.8, 'sky_brightness': 20.5, 'zeropoint': 25.0}
}

images = {}
for band, params in bands.items():
    psf = sim.generate_psf(fwhm=params['psf_fwhm'])
    image = sim.generate_image(stars, psf, sky_brightness=params['sky_brightness'])
    images[band] = image
    print(f"{band}波段图像生成完成: 均值={image.mean():.2f} ADU")
```

### 时域模拟（变星）
```python
import numpy as np
from freebirdcal.astro_simulator import AstronomicalSimulator

sim = AstronomicalSimulator(exposure_time=60.0)  # 60秒曝光

# 生成基础恒星
stars = sim.generate_stars(num_stars=500)

# 模拟变星光变
n_frames = 10
time_series = []

for i in range(n_frames):
    # 修改部分恒星的星等（模拟变星）
    stars_copy = stars.copy()
    n_variable = 10
    variable_indices = np.random.choice(len(stars['x']), n_variable, replace=False)
    
    # 随机光变幅度
    variation = np.random.uniform(-0.5, 0.5, n_variable)
    stars_copy['mag'][variable_indices] += variation
    stars_copy['flux'][variable_indices] = 10 ** (-0.4 * (stars_copy['mag'][variable_indices] - sim.zeropoint))
    stars_copy['flux_total'][variable_indices] = stars_copy['flux'][variable_indices] * sim.exposure_time
    
    # 生成图像
    psf = sim.generate_psf()
    image = sim.generate_image(stars_copy, psf)
    time_series.append(image)
    
    print(f"帧 {i+1}/{n_frames}: 生成完成")

print(f"时域模拟完成: {len(time_series)}帧图像")
```

## **故障排除**

### 常见问题

1. **内存不足**
   ```python
   # 减小图像尺寸或恒星数量
   sim = AstronomicalSimulator(image_size=512)  # 使用较小图像
   stars = sim.generate_stars(num_stars=100)    # 减少恒星数量
   ```

2. **通量不守恒警告**
   - 这是PSF卷积边界效应的正常现象
   - 可通过增大图像尺寸或减小PSF核尺寸来缓解

3. **文件保存失败**
   - 检查目录权限
   - 使用 `overwrite=True` 参数覆盖已存在文件
   - 检查磁盘空间

4. **数值溢出**
   - 减少曝光时间或使用较暗的星等范围
   - 检查 `min_mag` 和 `max_mag` 参数

## **FITS文件读取与分析**

`FITSReader` 类提供了便捷的FITS文件读取、分析和可视化功能。

### 基本用法
```python
from freebirdcal.astro_simulator import FITSReader

# 打开FITS文件（支持上下文管理器）
with FITSReader('observation.fits') as reader:
    # 获取文件信息
    info = reader.get_file_info()
    print(f"文件信息: {info}")
    
    # 打印摘要
    reader.print_summary()
    
    # 获取统计信息
    stats = reader.get_statistics()
    print(f"图像统计: 均值={stats['mean']:.2f}, 标准差={stats['std']:.2f}")
    
    # 显示图像
    reader.display_image(
        figsize=(12, 10),
        cmap='gray',
        stretch='log',  # 支持'linear', 'log', 'sqrt', 'asinh'
        percentile=99.5,
        title='天文观测图像'
    )
    
    # 保存为PNG
    reader.save_as_png('observation.png', dpi=300)
    
    # 绘制星表
    if reader.catalog_data is not None:
        reader.plot_catalog(
            x_col='X_PIXEL',
            y_col='Y_PIXEL',
            mag_col='MAG',
            show_image=True,  # 在背景上显示图像
            title='星表叠加图'
        )
```

### FITSReader类主要方法

| 方法 | 说明 |
|------|------|
| `__init__(filename)` | 初始化，打开FITS文件 |
| `close()` | 关闭FITS文件 |
| `get_file_info()` | 获取文件基本信息（尺寸、数据类型、星表等） |
| `get_header_summary(max_keys=20)` | 获取头部信息摘要 |
| `get_catalog_summary(max_rows=10)` | 获取星表摘要 |
| `get_statistics()` | 获取图像统计信息（最小值、最大值、均值、百分位数等） |
| `print_summary()` | 打印文件摘要到控制台 |
| `display_image(...)` | 显示图像，支持多种拉伸方式和颜色映射 |
| `save_as_png(output_path, **kwargs)` | 保存图像为PNG文件 |
| `plot_catalog(...)` | 绘制星表数据，可叠加在背景图像上 |

### 高级用法示例

#### 批量处理多个FITS文件
```python
import glob
from freebirdcal.astro_simulator import FITSReader

# 批量处理所有FITS文件
fits_files = glob.glob('observations/*.fits')

for fits_file in fits_files:
    print(f"\n处理文件: {fits_file}")
    
    with FITSReader(fits_file) as reader:
        # 获取基本信息
        info = reader.get_file_info()
        
        # 生成报告
        report = f"""
文件: {fits_file}
图像尺寸: {info['image_shape']}
星表行数: {info.get('catalog_rows', 0)}
曝光时间: {info.get('EXPTIME', 'N/A')}秒
增益: {info.get('GAIN', 'N/A')} e-/ADU
        """
        print(report)
        
        # 保存图像缩略图
        output_png = fits_file.replace('.fits', '_thumbnail.png')
        reader.save_as_png(output_png, figsize=(6, 5), dpi=100)
```

#### 数据提取与分析
```python
from freebirdcal.astro_simulator import FITSReader
import numpy as np

with FITSReader('science_data.fits') as reader:
    # 获取原始数据
    image_data = reader.image_data
    catalog = reader.catalog_data
    
    # 自定义分析
    if catalog is not None and 'FLUX' in catalog.colnames:
        fluxes = catalog['FLUX']
        
        print(f"通量统计:")
        print(f"  最小值: {np.min(fluxes):.2e} e-/s")
        print(f"  最大值: {np.max(fluxes):.2e} e-/s")
        print(f"  中位数: {np.median(fluxes):.2e} e-/s")
        print(f"  平均值: {np.mean(fluxes):.2e} e-/s")
        
        # 计算信噪比
        if 'FLUX_ERR' in catalog.colnames:
            snr = fluxes / catalog['FLUX_ERR']
            print(f"  平均信噪比: {np.mean(snr):.2f}")
    
    # 提取头部特定信息
    if reader.image_hdu is not None:
        header = reader.image_hdu.header
        if 'TELESCOP' in header:
            print(f"望远镜: {header['TELESCOP']}")
        if 'INSTRUME' in header:
            print(f"仪器: {header['INSTRUME']}")
```
