# 高级地形生成与可视化工具

本模块提供以下功能：
1. **构造抬升场生成**：模拟板块构造运动，生成空间变化的抬升速率场  
2. **初始地形生成**：基于 Simplex 噪声生成具备自然起伏的初始地形  
3. **水力侵蚀模拟**：使用 Landlab 组件模拟河流侵蚀、沉积物扩散及洼地填充  
4. **微观细节增强**：叠加多尺度分形噪声，增加岩石纹理和微小起伏  
5. **河流网络提取**：基于流量累积阈值识别河流分布  
6. **纹理权重图（Splatmap）生成**：根据海拔、坡度、河流分布自动生成四通道纹理权重图  
7. **格式转换与导出**：输出 Unity 兼容的 16 位 RAW 高度图和 PNG 格式 Splatmap  
8. **可视化与结果保存**：自动生成地形演化过程图、坡度图、河流网络图等

## **类说明**

### 一、 TerrainGenerator 类

#### 1. 功能
封装完整的地形生成流水线，支持构造抬升、侵蚀模拟、细节增强及纹理权重图生成，并提供便捷的输出和可视化接口。

#### 2. 初始化
```python
def __init__(self, shape=(513, 513), dx=10.0, seed=42)
```

**参数**：
- `shape` (tuple): 地形网格尺寸 `(nx, ny)`，推荐使用 `2^n+1` 形式（如 513、1025），便于 Unity 导入。
- `dx` (float): 网格间距（米），决定地形的实际空间尺度。
- `seed` (int): 随机种子，用于控制所有随机过程的可重复性。

#### 3. 主要方法

##### `generate_tectonic_field() -> np.ndarray`
生成空间变化的构造抬升速率场（单位：米/年）。  
内部使用 Voronoi 图划分板块，并叠加分形噪声使过渡自然。

##### `initial_topography(scale=300, octaves=7) -> np.ndarray`
生成初始地形（海拔范围 0~5000 米）。  
基于 Simplex 噪声构建分形地形，`scale` 控制宏观特征大小，`octaves` 控制细节层次。

##### `run_erosion(total_time=300000, dt=100, output_interval=500) -> np.ndarray`
运行水力侵蚀模拟（需已安装 Landlab）。  
- `total_time`：模拟总时间（年）  
- `dt`：时间步长（年）  
- `output_interval`：进度打印间隔（步数）  

##### `add_micro_details(amplitude=30, scale=80) -> np.ndarray`
叠加微观细节（小尺度分形噪声）。  
- `amplitude`：细节起伏幅度（米）  
- `scale`：噪声尺度（像素单位）

##### `extract_rivers(threshold_m2=500000) -> np.ndarray`
提取河流网络掩膜。  
使用 Landlab 的流量累积结果，将汇水面积大于 `threshold_m2`（平方米）的单元格标记为河流。若 Landlab 不可用，则使用高程局部最低点作为简单替代。

##### `compute_slope(z=None) -> np.ndarray`
计算地形坡度（度）。  
若不传入 `z`，则使用当前 `final_z`。

##### `generate_splatmap(rivers_mask=None) -> np.ndarray`
根据海拔、坡度、河流掩膜生成四通道纹理权重图（RGBA）。  
纹理分配规则：
- 通道 0 (R)：低海拔草地（海拔 < 1500 m 且坡度 < 30°）  
- 通道 1 (G)：中海拔岩石（1500~3500 m 且坡度 < 45°）  
- 通道 2 (B)：高海拔雪地（海拔 ≥ 3500 m）  
- 通道 3 (A)：河流/沙地（河流掩膜区域）  
若未提供 `rivers_mask`，则使用当前 `self.rivers_mask`。

##### `save_raw(filename, z=None)`
保存为 16 位 RAW 高度图（Unity 地形导入格式）。  
自动将高程归一化到 0~1 后映射至 0~65535。

##### `save_splatmap_png(filename)`
将 splatmap 保存为 RGBA PNG 图像。

##### `visualize(save_plots=False, prefix="terrain")`
生成可视化图表，包括：
- 初始地形、侵蚀后地形、最终地形（三列对比）
- 坡度图
- 河流网络掩膜图
- Splatmap RGB 通道示意图  
若 `save_plots=True`，则保存为 `prefix_visualization.png`。

##### `run_full_pipeline(total_time=300000, dt=100, micro_amplitude=30, micro_scale=80, river_threshold=500000, save_plots=True, out_prefix="terrain")`
执行完整生成流程（依次调用上述方法），并保存所有输出文件。  
各参数含义同上。

## **示例**

### 基础用法（一键生成）
```python
from terrain_generator import TerrainGenerator

# 创建生成器实例
generator = TerrainGenerator(shape=(513, 513), dx=10.0, seed=42)

# 执行完整流程
generator.run_full_pipeline(
    total_time=300000,          # 模拟 30 万年
    dt=100,                     # 时间步长 100 年
    micro_amplitude=30,         # 微观细节幅度 30 米
    micro_scale=80,             # 细节尺度
    river_threshold=500000,     # 河流汇水面积阈值 0.5 km²
    save_plots=True,            # 保存可视化图像
    out_prefix="my_terrain"     # 输出文件名前缀
)
```

### 分步执行（自定义流程）
```python
generator = TerrainGenerator(shape=(1025, 1025), dx=5.0, seed=123)

# 仅生成初始地形并保存
generator.initial_topography(scale=400, octaves=8)
generator.save_raw("initial.raw")

# 单独运行侵蚀并保存
generator.run_erosion(total_time=500000, dt=200)
generator.save_raw("eroded.raw")

# 添加细节后生成 splatmap
generator.add_micro_details(amplitude=20, scale=100)
generator.generate_splatmap()
generator.save_splatmap_png("splatmap.png")
```

### 获取中间数据用于自定义处理
```python
generator = TerrainGenerator()
generator.run_full_pipeline()

# 获取最终高程数组（单位：米）
elevation = generator.final_z

# 获取 splatmap 权重数组（形状 (nx, ny, 4)，0~1 浮点数）
weights = generator.splatmap

# 获取河流掩膜（布尔数组）
rivers = generator.rivers_mask

# 自行保存为其他格式（如 GeoTIFF）
import rasterio
with rasterio.open("output.tif", 'w', driver='GTiff', height=elevation.shape[0],
                   width=elevation.shape[1], count=1, dtype=elevation.dtype) as dst:
    dst.write(elevation, 1)
```

## **依赖与注意事项**

### 必须安装的库
- `numpy`
- `scipy`
- `noise`
- `Pillow`
- `matplotlib`
- `landlab`（用于侵蚀模拟，若不可用则跳过侵蚀步骤）

安装命令：
```bash
pip install numpy scipy noise pillow matplotlib landlab
```

### 运行环境
- 推荐使用 Python 3.8 及以上版本。
- Landlab 的 D8 流向算法对内存有一定要求，`shape` 较大时（如 1025×1025）建议使用内存充足的机器。

### 输出文件说明
| 文件                 | 格式         | 用途                                                                 |
|----------------------|--------------|----------------------------------------------------------------------|
| `*_heightmap.raw`    | 16 位 RAW    | Unity 地形高度图，通过 `Terrain -> Import Raw` 导入                  |
| `*_splatmap.png`     | RGBA PNG     | 纹理权重图，在 Unity 中使用 `TerrainData.SetAlphamaps` 加载          |
| `*_visualization.png`| PNG          | 地形演化及属性可视化图，便于观察生成效果                              |

### Unity 导入 Splatmap 示例（C# 脚本）
```csharp
using UnityEngine;

public class ApplySplatmap : MonoBehaviour
{
    public Terrain terrain;
    public Texture2D splatmapTexture;

    void Start()
    {
        TerrainData data = terrain.terrainData;
        int w = data.alphamapWidth;
        int h = data.alphamapHeight;
        float[,,] splatmap = new float[h, w, 4];

        Color[] pixels = splatmapTexture.GetPixels();
        for (int y = 0; y < h; y++)
            for (int x = 0; x < w; x++)
            {
                Color c = pixels[y * w + x];
                splatmap[y, x, 0] = c.r;
                splatmap[y, x, 1] = c.g;
                splatmap[y, x, 2] = c.b;
                splatmap[y, x, 3] = c.a;
            }
        data.SetAlphamaps(0, 0, splatmap);
    }
}
```
将脚本挂载到地形对象，并指定生成的 splatmap 纹理即可完成纹理混合。
